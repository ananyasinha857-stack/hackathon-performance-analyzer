# Hackathon Performance Analyzer

A Python-based analytics tool that evaluates hackathon performance across multiple events and generates visual insights.

## Features

- Event-wise performance breakdown
- Team progression tracking
- Heatmaps and comparative visualizations

## Technologies

- Python
- Pandas
- Matplotlib
- Seaborn

## Output

The analyzer generates:

- Per-event breakdown
- Journey & heatmap visualization
- Structural insights report

## Run

pip install pandas matplotlib seaborn

python analyzer.py
